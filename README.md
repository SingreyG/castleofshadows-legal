# Castle of Shadows — Legal

Public legal pages for the **Castle of Shadows** mobile game by Singreybuilds.

Served via GitHub Pages at:

- https://singreyg.github.io/castleofshadows-legal/

Pages:

- `index.html` — landing
- `privacy.html` / `privacy-tr.html` — Privacy Policy (EN / TR)
- `terms.html` / `terms-tr.html` — Terms of Service (EN / TR)

## Updating the policy

1. Edit the relevant `.html` file (English first, then Turkish — keep them in
   sync)
2. Update the **Last updated** / **Son güncelleme** date at the top
3. Commit + push — GitHub Pages will redeploy within a minute

## Why a separate public repo

The game lives in a private repo. Legal pages need to be publicly reachable
(App Store / Play Console submission forms link to them, regulators may
verify them). Splitting them keeps the game's source closed while leaving
the legal surface open and immutable on its own URL.
